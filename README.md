# おたよりBOX

子育て中の保護者が、園・学校・習い事のおたよりを
子ども別・月別に管理できる Google Apps Script 完結の Web アプリです。

---

## 概要

紙のおたより・学校アプリのPDF・LINEで届くお知らせ──
複数経路に散らばるおたよりを、撮影またはPDF保存するだけで
「子ども別・月別」に自動で整理します。

---

## ターゲットユーザー

- 30〜40代の共働きママ
- 子ども3人を持つ家庭
- ITが得意ではないユーザー

---

## 解決する問題

| 困りごと | 解決策 |
|---|---|
| おたよりが紙・PDF・LINEに散らばって探せない | スクショ／PDF を3クリック以内で保存 |
| 子ども別に整理するのが面倒 | 子ども選択＋日付で Drive に自動振り分け |
| 過去のおたよりを見返すのが大変 | 月別グループ＋プルダウンで瞬時に絞り込み |
| 重要なお知らせを見失う | ピン留めで一覧上部に固定表示 |

---

## 技術スタック

| 種別 | 技術 |
|---|---|
| バックエンド | Google Apps Script（GAS） |
| 画面表示 | GAS HTML Service |
| フロントエンド | Vanilla JavaScript（HTMLに直接記述） |
| データベース | Google Spreadsheet |
| ファイル保存 | Google Drive |
| アイコン | Lucide Icons（CDN） |
| フォント | Poppins / Noto Sans JP（Google Fonts CDN） |
| PDF表示 | PDF.js（CDN） |

外部DB・フレームワーク・npmパッケージ・ビルドツールは一切不使用。
すべて Google Workspace 無料枠で動作します。

## 実装済み機能

### MVP（F-01〜F-11）

- 初回セットアップ（1クリックでシート・Driveフォルダ自動作成）
- 接続状態確認・エラー対処ヘルプ
- 子ども管理（登録・編集・削除・最大10人）
- ホーム画面（子ども一覧カード）
- おたより登録（子ども選択→画像選択→保存確認の3ステップ）
- おたより一覧（子ども別タブ＋月別グループ＋サムネイル）
- おたより拡大表示モーダル
- レスポンシブUI（スマホ・タブレット・PC）

### v1.0（F-12〜F-17）

- 最近のおたより表示（ホーム画面・全子ども横断・直近5件）
- 複数枚同時登録（最大5枚・進捗バー）
- PDF登録対応
- 月フィルター強化（プルダウン式）
- フルスクリーン表示・スワイプ移動
- ピン留め（一覧上部に固定表示）
- おたより削除（Driveゴミ箱送り・30日復元可能）

### 追加機能

- 子どもごとのバッジカラー選択（10色から選択）
- おたよりに任意タイトル付与（登録時／後追い編集対応）
- 月フィルター降順ソート＋プルダウン化

## デプロイ方法（GAS WebAppとして公開）

### 1. スクリプトファイルを GAS プロジェクトに配置

1. Google Apps Script（script.google.com）を開く
2. 新しいプロジェクトを作成
3. このリポジトリ内の以下ファイルをスクリプトエディタに同名でコピー：
   - `Code.gs` / `Constants.gs` / `Utils.gs` / `Setup.gs`
   - `SpreadsheetService.gs` / `DriveService.gs` / `ChildService.gs` / `NoticeService.gs` / `UIService.gs`
   - `index.html` / `css.html` / `js.html`
   - `home.html` / `upload.html` / `children.html` / `notices.html` / `settings.html` / `setup_required.html`

### 2. ウェブアプリとしてデプロイ

1. スクリプトエディタ → デプロイ → 新しいデプロイ
2. 種類：ウェブアプリ
3. 次のユーザーとして実行：自分
4. アクセスできるユーザー：自分のみ（友人共有時は対象のGoogleアカウントを追加）
5. デプロイ → 公開URLをコピー

### 3. 公開URLを開く

公開URLをブラウザ（スマホ可）で開き、画面の案内に従って初回セットアップを実行する。

## 使い方

### 初回セットアップ

1. 公開URLを開くと「セットアップが必要です」画面が表示される
2. スクリプトエディタを開き、メニューバーの **おたよりBOX → 初回セットアップ実行** をクリック
3. 必要なシート（子ども一覧／おたより一覧／システム設定）と Drive フォルダ（おたよりBOX）が自動作成される
4. 公開URLを再読込

### 子ども登録

1. 下部ナビゲーション「子ども」をタップ
2. 「子どもを追加」→ 名前を入力 → 保存
3. 詳細カードで好きなバッジカラーを選択（10色から）

### おたより登録

1. 下部ナビゲーション中央のカメラアイコンをタップ
2. **STEP1**：対象の子どもを選択
3. **STEP2**：画像または PDF を選択（最大5枚同時可）
4. **STEP3**：対象年月・任意タイトルを確認 → 保存
5. Drive の `おたよりBOX / [子ども名] / [年月]` フォルダに自動保存される

### おたよりを見る

1. 下部ナビゲーション「一覧」をタップ
2. 子どもタブで絞り込み・月プルダウンで月絞り込み
3. サムネイルをタップでフルスクリーン表示
4. 左右スワイプで前後のおたよりへ移動
5. ピンアイコンで一覧上部に固定表示
