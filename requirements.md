## 機能要件

| ID | 機能名 | 説明 | 優先度 | 受け入れ条件（AC） |

| F-05 | 子ども削除 | 登録済みの子どもを削除する | 中 | 「本当に削除しますか？」の確認ダイアログ後に削除される |
| F-06 | ホーム画面 | 子ども一覧カードを表示するアプリの起点画面 | 高 | 子どもカードと登録ボタンが表示される |
| F-07 | おたより登録（1枚） | 子ども選択→画像・PDF選択→保存確認の3ステップで登録する | 高 | 3クリック以内で保存が完了しDriveの子ども別・月別フォルダに格納される |
| F-08 | おたより一覧 | 子ども別タブ切替＋月別グループでサムネイル表示する | 高 | 子どもタブを選択すると該当おたよりのみ月別グループで表示される |
| F-09 | おたより拡大表示 | サムネイルタップで画像・PDFをモーダル表示する | 高 | タップで全画面表示になり子ども名・登録年月（曜日付き）が表示される |
| F-10 | レスポンシブUI | スマホ最優先・タブレット・PC対応のBottomNavigation構成 | 高 | スマホ・タブレット・PCいずれでも主要機能が2クリック以内で操作できる |
| F-11 | エラー対処ヘルプ | よくあるトラブルと対処法を表示する | 高 | トラブル一覧が表示され診断ページへのリンクが機能する |

## 非機能要件

### パフォーマンス
- 画面初期表示：3秒以内
- 画像アップロード（1枚）：10秒以内
- 一覧取得（20件以下）：3秒以内

### セキュリティ
- 認証：Googleアカウント（GAS WebAppの権限設定で制御）
- 画像データはSpreadsheetに保存しない。必ずGoogle Driveに保存する
- Script Propertiesにシークレット情報を保管。コード内へのハードコーディング禁止
- Driveフォルダは所有者のGoogleアカウントでのみアクセス可能

### 可用性
- GoogleサービスのSLAに依存（個人利用のため高SLAは不要）
- GAS無料枠の制限内で動作すること（実行時間：6分/回・1日合計90分）

### スケーラビリティ
- 1家族・1Googleアカウントでの単独利用を前提とする
- 子ども登録数：最大10人
- 対応ファイル形式：JPEG・PNG・HEIC（MVP）／PDF（v1.0）・最大10MB/枚

### ユーザビリティ
- 説明書なしで初回利用から登録操作を完了できること
- スマホ片手操作・タップ中心のUI
- 主要機能まで2クリック以内

### 保守性・拡張性
- 将来のAI要約機能追加を見据えたカラム設計（summary・category・eventDate・itemsを予約列として確保）
- 不足カラムを自動追加するマイグレーション機能をSetup.gsに実装する
- 定数はすべてConstants.gsで一元管理。ハードコーディング禁止

## 制約条件

### 予算
- すべて無料枠で完結（Google Workspace 無料版）
- 外部有料サービス・APIの使用禁止

### 技術スタック
- Google Apps Script（バックエンド）
- HTML Service（画面表示）
- Vanilla JavaScript（フロントエンド）
- Google Spreadsheet（データベース）
- Google Drive（ファイル保存）
- Lucide Icons CDN・Google Fonts CDN（UI）

### 使用禁止技術
- React / Next.js / Vue.js
- Firebase / Supabase
- Node.js / npmパッケージ
- 外部データベース
- ビルド環境が必要なもの

### コーディング規約（詳細はCLAUDE.mdを参照）
- `var` / `function` スタイルで統一（`let` / `const` 不使用）
- 日本語コメントを論理ブロックごとに記載
- 全関数にtry-catchを実装
- 処理完了時は日本語のトーストまたはアラートで通知
- 定数はConstants.gsで一元管理

## スコープ

### In Scope（作るもの）
- おたよりのスクショ・PDFを子ども別・月別に保存する機能
- 保存したおたよりを一覧で確認・拡大表示する機能
- 子ども情報の管理（追加・編集・削除）
- 初回セットアップの自動化・接続状態確認・エラー対処ヘルプ

### Out of Scope（作らないもの）
- AIによるおたより内容の自動解析・要約（v2.0以降）
- カレンダー連携・Gmailリマインド通知（v2.0以降）
- 検索・タグ分類機能（v2.0以降）
- 複数家族・複数アカウントでの共有
- iOSアプリ・Androidアプリのネイティブ版
- 有料サーバー・有料DBの使用
